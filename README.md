# **Application Overview**:
An application designed to help users find local grocery deals through physical mailers or a digital equivalent. It will also suggest recipes based on the selected ingredients.

## **Subscription Service**:
The app will offer better options compared to traditional meal kits. 

### **Accessibility:**
The service will be free for individuals in need. I will need to determine the best method to verify eligibility and to charge users with better financial resources.

**Price Tracking:**
The application will track prices for various grocery items across the country over time.

**Name Ideas:**
- Chow Bella

**Development Phases:**
1. Build a frontend that locates the user and identifies grocery stores in their area.
2. Scrape data from the most recent mailers for those stores and create a user interface (UI) for making a grocery list based on that data.
3. Based on items marked for purchase by the user, suggest recipes (using language models or additional data scraped from cooking sites). Update the shopping list with any necessary additions for preparing the suggested dishes.
4. Implement a system for tracking item prices over time and highlighting special deals.
